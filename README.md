# Rock Rush ⛏️

A fast-paced arcade mining game. Chip down the endless rock pillar, dodge the
crystal clusters, and don't let the timer run out — it drains faster the higher
your score climbs.

**Play it here: https://andrewlocke2001-debug.github.io/rock-rush/**

## Controls

- **Tap / click** the left or right half of the screen to mine on that side
- Or use the **← →** arrow keys (or **A** / **D**)
- Dodge the purple crystals — switching into one, or letting one drop onto you, ends the run
- Every hit refills a little of the timer bar; your best score is saved locally

## Running locally

It's a single self-contained `index.html` — no build, no dependencies.
Open the file directly in a browser, or serve the folder:

```
npx serve .
```
